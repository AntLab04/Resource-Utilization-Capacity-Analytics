# Resource Utilization & Capacity Analytics
 
## Problem Statement
Organizations often face challenges in balancing workforce capacity, leading to under-utilization, burnout, and inefficiencies. This project analyzes resource utilization data to identify capacity risks and optimization opportunities.
 
## Tools Used
- Microsoft Excel
- Power BI
 
## Key Analysis
- Average utilization tracking
- Under- and over-utilization identification
- Team-wise and client-wise utilization analysis
- Monthly utilization trend analysis
- Utilization band distribution
 
## Key Insights
- Overall utilization averages ~67%, indicating available idle capacity.
- HR Ops and Client C show higher workload concentration.
- A significant portion of resources remain under-utilized, while some show over-utilization risk.
- Opportunities exist for workload rebalancing and improved capacity planning.
 
## Outcome
The analysis highlights actionable insights for optimizing workforce allocation, reducing burnout risk, and improving overall operational efficiency.
